Para ejecutar el proyecto abra la carpeta src en vscode, ubiquese en el archivo "./vista/principal.java",
despues sobre el archivo dar click izquierdo y seleccionar la opcion ejecutar, para terminar el proceso 
presione el comando Ctrl+C.

Si desea probra con otro texto ubiquese en el archivo "./vista/principal.java", en dicho archivo podra modificar el texto y ejecutar nuevamente para ver los resultados.
